## 🎯 **USANDO O SISTEMA**

### **🎲 Gerando Predições**

#### **Predição Simples**
1. **Selecione o modelo** desejado no dropdown
2. **Escolha o concurso** alvo (próximo concurso)
3. **Clique em "Gerar Predição"**
4. **Aguarde o resultado** (1-3 segundos)

#### **Modelos Disponíveis**
```
🎵 MetronomoModel
   - Tipo: Frequencista (baseline)
   - Performance: 60-65%
   - Melhor para: Padrões temporais

🔄 AntiFrequencySimple
   - Tipo: Anti-frequencista básico
   - Performance: 63-66%
   - Melhor para: Inversão de tendências

💰 StatisticalDebt
   - Tipo: Dívida estatística
   - Performance: 64-70%
   - Melhor para: Análise matemática

📈 SaturationModel
   - Tipo: Saturação por RSI
   - Performance: 64-68%
   - Melhor para: Análise técnica

🧠 MetaLearning
   - Tipo: Ensemble inteligente
   - Performance: 75%+
   - Melhor para: Máxima performance
```

### **📊 Interpretando Resultados**

#### **Formato de Saída**
```
Predição: 01, 03, 05, 07, 09, 11, 13, 15, 17, 19, 21, 23, 25, 02, 04
Confiança: 67.3%
```

## 🎭 **FUNCIONALIDADES AVANÇADAS**

### **🔧 Configuração de Modelos**

#### **Acessando Configurações**
1. **Selecione o modelo** desejado
2. **Clique em "Configurar"**
3. **Ajuste os parâmetros** conforme necessário
4. **Salve as configurações**

#### **Parâmetros Principais**
```
MetronomoModel:
├── FatorRuido: 0.1 (0.0-1.0)
├── IntervaloBase: 5 (1-20)
└── TipoMetronomo: Simples|Alternado|CicloLongo

AntiFrequencySimple:
├── JanelaAnalise: 100 (50-500)
├── PesoTemporal: 0.7 (0.0-1.0)
└── ThresholdMinimo: 0.05 (0.01-0.20)

StatisticalDebt:
├── FatorAceleracao: 1.5 (1.0-3.0)
├── JanelaAnalise: 200 (100-1000)
├── PesoTemporal: 0.8 (0.0-1.0)
└── ThresholdDivida: 0.1 (0.01-0.50)

SaturationModel:
├── JanelaAnalise: 100 (50-300)
├── SaturationWeight: 0.4 (0.0-1.0)
└── PesoTemporal: 0.7 (0.0-1.0)
```

### **📈 Comparação de Modelos**

#### **Executando Comparação**
1. **Clique em "Comparar Modelos"**
2. **Aguarde a análise** (5-10 segundos)
3. **Visualize o relatório** comparativo
4. **Identifique o melhor modelo** para o contexto

#### **Métricas de Comparação**
```
Performance Individual:
├── Accuracy: % de acertos
├── Confiança Média: Nível de certeza
├── Tempo de Execução: Velocidade
└── Correlação: Diversificação

Análise Ensemble:
├── Melhor Combinação: Modelos ideais
├── Correlação Cruzada: Independência
├── Performance Conjunto: Resultado final
└── Recomendação: Estratégia sugerida
```
